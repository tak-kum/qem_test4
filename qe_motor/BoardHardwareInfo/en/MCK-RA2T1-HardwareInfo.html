MCK-RA2T1 Renesas Flexible Motor Control Kit for RA2T1 MCU Group

RTK0EMA810S00020BJ

The MCK-RA2T1 is a development kit that enables easy evaluation of motor control using permanent magnet synchronous motors (brushless DC motors). With this product and the sample code and QE for Motor that can be downloaded from the website, you can start evaluating motor control using the RA2T1 right away.

Product Specification

    MCK-RA2T1
    • The CPU board is equipped with the RA2T1 as the motor control MCU.
    • Inverter board for 3-phase BLDC motor (48V/10A rating)
    • On-board debugger for flash programming of RA2T1
    • Supports 1-shunt/3-shunt current sensing
    • Overcurrent detection
    • Supports the motor control development support tool, Renesas Motor Workbench
    • Motor control evaluation and debugging can be performed safely with the communication board which isolates PC and the CPU board electrically
    • Supports Hall sensors, encoders, and inductive position sensors
    • Included items
      • RA2T1 CPU board (MCB-RA2T1)
      • Inverter board (MCI-LV-1)
      • Communication board (MC-COM)
      • Permanent magnet synchronous motors
      • Accessories (cables, standoffs, etc.)
